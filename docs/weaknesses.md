# Represention errors

* [overprecision](https://en.wikipedia.org/wiki/False_precision), e.g. time value is represented with a larger granularity than its [datetime storage granularity](https://dfdatetime.readthedocs.io/en/latest/sources/Date-and-time-values.html#terminology)
* [underprecision](https://en.wikipedia.org/wiki/Round-off_error), e.g. time value is represented with a smaller granularity than its [datetime storage granularity](https://dfdatetime.readthedocs.io/en/latest/sources/Date-and-time-values.html#terminology)

# Repeatability errors

* non-transmittable (internal to implementation) values, e.g. TSK metadata address

# Observation errors

* https://en.wikipedia.org/wiki/Observational_error
* https://www.panarchy.org/ackoff/observation.html

* Fog Distortion (Not seeing what is there): Missing data (commission error).
* Mirage Distortion (Seeing what is not there): False observations (ommission error).
